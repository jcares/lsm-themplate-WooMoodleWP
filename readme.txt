LSM WooMoodle WP Theme
=======================

Theme Description:
- Progressive futuristic design for e-learning with Moodle, WooCommerce and Elementor support.
- Includes pages: Home, About, Courses, Contact with templates under templates/.
- Assets in /images/, /css/, /js/ with glassmorphism style.

Installation:
1. Copy this folder to wp-content/themes/lsm-themplate-WooMoodleWP
2. Activate theme in WP admin.
3. Go to Appearance -> Customize to set brand, colors, social links.
4. Optional: install WooCommerce and Elementor.

Plugin:
- Includes plugin "moodle-woocommerce-sync" in root (for dev) to import Moodle courses to WooCommerce products.
- Configure Moodle URL and token in WP admin under Settings.
- Supports manual sync and cron sync every hour.

Required:
- WordPress 6.0+
- PHP 7.4+
- WooCommerce 7.0+
